Sobre a Calculadora de Média
Visão Geral
A Calculadora de Média é uma aplicação simples e intuitiva desenvolvida para facilitar o cálculo da média de notas escolares. Criada utilizando Electron, a aplicação combina tecnologias web modernas com a capacidade de ser executada como um aplicativo desktop.
Motivação
A ideia para este projeto surgiu da necessidade de uma ferramenta prática e acessível para estudantes e professores que precisam calcular a média de notas de forma rápida e eficiente. Com uma interface amigável e fácil de usar, a Calculadora de Média permite que qualquer pessoa insira suas notas e obtenha o resultado da média instantaneamente.
Funcionalidades Principais
Inserção de Notas: O usuário pode inserir até quatro notas diferentes.
Cálculo Instantâneo: Com um único clique, a média das notas é calculada e exibida na tela.
Interface Amigável: O design é simples e claro, com um layout que facilita a interação do usuário.
Tecnologias Utilizadas Electron: Framework que permite o desenvolvimento de aplicações desktop utilizando tecnologias web como HTML, CSS e JavaScript. HTML e CSS: Para estruturação e estilização da interface do usuário. JavaScript: Para a lógica de cálculo e manipulação do DOM.
Como Funciona
A Calculadora de Média recebe quatro valores numéricos através de um formulário. Quando o usuário clica no botão "Calcular Média", a aplicação realiza o cálculo da média aritmética desses valores e exibe o resultado na tela.
Sobre o Desenvolvedor
A Calculadora de Média foi desenvolvida por Susan Marques, uma entusiasta da tecnologia com paixão por criar soluções práticas e eficientes. Este projeto é uma demonstração das possibilidades oferecidas pelo Electron para o desenvolvimento de aplicações desktop simples e funcionais.
